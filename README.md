# This WeatherApp allows the user to
view a list of all weather informations from the API
* The city name
* The current weather description (e.g. cloudy)
* The current temperature
* humidity
* wind speed
* Feels like
To run locally
1. Clone the repository
2. Run npm i
Dependencies
* axios
* react
3. Run npm start
